# Práctica 4: Página web estática

Página web estática sobre un carrito de compra de ocio y eventos, desarrollada con HTML y CSS.

## Contenido

La web consta de 4 páginas interconectadas:

- **Inicio (index.html)**: Portada con presentación del servicio, recursos multimedia (imagen, vídeo, audio) y agenda.
- **Productos (productos.html)**: Catálogo de eventos con tabla y tarjetas de producto.
- **Carrito (carrito.html)**: Resumen del carrito con estructura de compra y totales.
- **Contacto (contacto.html)**: Formulario de contacto.

## Elementos HTML empleados

- Títulos con jerarquía h1, h2 y h3
- Párrafos descriptivos
- Tablas (agenda y catálogo)
- Multimedia (imagen, vídeo y audio)
- Formularios web

## Estilo

La web tiene un diseño personal y uniforme aplicado con CSS. Incluye una paleta de colores consistente, componentes reutilizables y adaptación responsive para diferentes tamaños de pantalla.
